# 프로젝트 세팅

###  1. [p2ach_android_library_init.sh](https://github.com/p2achAI/P2achAndroidLibrary/blob/main/script/p2ach_android_library_init.sh) 파일을 다운로드 
###  2. 로컬 저장소의 Root가 될 폴더에서 터미널 실행
###  3. 권한 설정 chmod +x p2ach_android_library_init.sh
###  4. 실행 ./2ach_android_library_init.sh

이후 AndroidStudio로 Open -> Build 
## Build 안되면 관리자에게 문의하세요.
